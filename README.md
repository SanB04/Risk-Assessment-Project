**Project Overview:**

This project is a simulated Information Security risk assessment for PayLink Digital Solutions. The objective was to identify critical information assets, evaluate potential risks affecting the organization’s infrastructure and document the findings through a structured risk register. The assessment follows risk management principles used in frameworks such as ISO/IEC 27001.

**Project Activities:**

•	Defined the ISMS scope for PayLink’s Digital payment system

•	Identified and documented important information assets, including applications, data systems, and infrastructure

•	Created an asset inventory that contains important assets, ownership, classification, and system dependencies

•	Conducted risk assessment by identifying potential infrastructure, cybersecurity, operational and compliance risks

•	Evaluated risks using a likelihood and impact methodology

•	Created a risk register documenting risk descriptions, treatment approaches, risk owners and monitoring status

**Key Deliverables:**

The project include the following deliverables:

•	ISMS Scope Document

•	Asset Inventory Spreadsheet

•	Risk Register Spreadsheet

•	Risk Register Documentation
